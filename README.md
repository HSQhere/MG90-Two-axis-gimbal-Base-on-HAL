# MG90-Two-axis-gimbal-Base-on-HAL

基于 **STM32F103** 的双轴摇杆控制云台项目。通过 ADC 采集摇杆模拟信号，利用 **DMA** 提高采样效率，并结合 **内部参考电压校准** 技术，实现对两个 MG90 舵机（Yaw & Pitch）的精准平滑控制。

## 🌟 项目亮点
* **DMA 多通道采样**：自动将 3 路 ADC 数据（X轴、Y轴、内部参考电压）搬运至内存，不占用 CPU。
* **电压自校准**：通过监测内部 1.2V 参考电压，消除因供电波动引起的角度偏移。
* **死区逻辑 (Deadzone)**：有效过滤摇杆中位抖动，防止舵机在静止时震动。
* **VOFA+ 调试支持**：代码内置 FireWater 协议，支持实时数据可视化。

## 🔌 硬件连接表

| 硬件模块 | 信号线 | STM32 引脚 | 备注 |
| :--- | :--- | :--- | :--- |
| **双轴摇杆** | VRX (X轴) | PB0 | ADC1_CH8 |
| **双轴摇杆** | VRY (Y轴) | PB1 | ADC1_CH9 |
| **Yaw 舵机** | PWM 信号 | PA0 | TIM2_CH1 |
| **Pitch 舵机** | PWM 信号 | PA1 | TIM2_CH2 |

## ⚙️ 核心参数
* **PWM 频率**: 50Hz (周期 20ms)
* **脉宽范围**: 500us (0°) ~ 2500us (180°)
* **ADC 范围**: 0 - 4095 (12-bit)

## 🚀 核心逻辑
```c
// 角度映射公式
float pulse_us = 500 + (angle / 180.0f) * 2000;
```

## 📈 调试说明
本项目输出格式兼容 VOFA+ (FireWater)。
在串口调试助手或 VOFA+ 中打开波形图，发送格式为：
raw_x, raw_y, angle_x, angle_y\n

## 🛠️ 如何使用
* 使用 STM32CubeMX 打开 .ioc 文件检查配置。
* 使用 Keil uVision 5 或 STM32CubeIDE 编译下载。
* 确保舵机使用 5V 供电，单片机使用 USB 或 3.3V 供电。

**注意：如果舵机在极值点（4000附近）转动不畅，请检查舵机机械限位，并在代码中适当缩小 SERVO_MAX_US 的值。**
